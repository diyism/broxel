1. Understanding Modes

Brow6el has four distinct modes, each designed for different tasks:

  • [S] STANDARD - Single-key commands for navigation and browser controls (default)
  • [I] INSERT - All keys pass through to web forms and input fields
  • [M] MOUSE - Keyboard-based mouse emulation for clicking links and elements
  • [V] VISUAL - Select and copy text from web pages
💡 Tip: The current mode is always shown in the status bar at the bottom of the screen. Press ESC in any mode to return to STANDARD mode.

🎯 Check Your Status Bar

Look at the bottom of your screen right now. You should see [S] indicating you're in STANDARD mode.

2. STANDARD Mode - Navigation & Commands

This is your home base. From STANDARD mode, you can navigate pages, manage bookmarks, and access all browser features with single-key commands.

Page Navigation

  • hjkl Move left, down, up, right (like Vim arrow keys)
  • t Scroll up (think "top")
  • g Scroll down (think "go down")
Try scrolling this page using j (down) and k (up). Get a feel for the rhythm!

URL & Page Management

  • u Navigate to URL (opens URL input dialog)
  • p Previous page in history (back)
  • n Next page in history (forward)
  • r Reload current page
  • d Add current page to bookmarks
  • b Open bookmarks menu
Press u to open the URL dialog. Type a URL and press Enter to navigate. Try example.com!

Advanced Navigation

  • f Hint mode - show link labels for keyboard navigation
  • e Enter MOUSE mode for precise clicking
  • v Enter VISUAL mode for text selection

Developer & Advanced

  • c Toggle JavaScript console
  • s User scripts menu
  • y Toggle auto-inject user scripts
  • z Toggle tiled rendering (sixel only)
  • Z Force full screen redraw

Mode Switching

  • i Enter INSERT mode (for forms)
  • x Exit browser

3. MOUSE Mode - Keyboard Mouse Control

MOUSE mode lets you control a virtual mouse cursor using your keyboard. This is perfect for clicking buttons, interacting with complex web apps, or when hint mode doesn't capture all clickable elements.

💡 Activation: Press e from STANDARD mode. A colored cursor will appear on screen. The color indicates speed mode.

Cursor Movement

  • hjkl Move cursor left, down, up, right
  • q Toggle precision mode (slow, accurate - blue cursor)
  • f Toggle fast mode (quick movement - red cursor)

Actions

  • SPACE Click at cursor position
  • ENTER Click at cursor position
  • r Toggle drag mode (for drag & drop)
  • g Grid mode - quick jump to screen regions

Exit

  • e Exit MOUSE mode back to STANDARD
  • ESC Exit MOUSE mode back to STANDARD
Try entering MOUSE mode with e, moving the cursor with hjkl, and clicking this button:

Grid Mode

Grid mode divides the screen into a 3×3 grid. Press g in MOUSE mode to activate, then press one of these keys to jump:

q w e
a s d
z x c

Each key corresponds to a screen position. By default, these keys match your keyboard layout for easy memorization.

4. INSERT Mode - Text Input

INSERT mode is where all your keypresses go directly to the webpage. This is essential for filling out forms, typing in search boxes, and interacting with web applications.

💡 How to use: Press i from STANDARD mode. The status bar will show [I]. Now you can type normally in any form field.
  • ESC Return to STANDARD mode

5. VISUAL Mode - Text Selection

VISUAL mode lets you select and copy text from web pages using keyboard navigation.

💡 Activation: Press v from STANDARD mode. The status bar shows [V].

Selection Movement

  • hjkl Move selection left, down, up, right
  • w Move to next word
  • b Move to previous word

Actions

  • y Yank (copy) selected text to clipboard
  • ESC Exit VISUAL mode back to STANDARD
Try selecting this text: Press 'v', navigate with 'hjkl', and press 'y' to copy! Then paste it somewhere to verify.

6. Advanced Features

Hint Mode (Link Navigation)

Hint mode is one of the most powerful features. Press f and all clickable elements get labeled with letters/numbers.

💡 Usage: Press f, then type the label shown next to the link you want to click. The link will be activated instantly!

🎯 Practice Links

Press f now and try clicking these links:

  • Jump to Modes section
  • Jump to Standard Mode
  • Jump to Tips section

Bookmarks

  • Press d to bookmark the current page
  • Press b to open bookmarks, use j/k to navigate, ENTER to open
  • Press x in bookmarks menu to delete selected bookmark

User Scripts

Inject custom JavaScript into web pages for enhanced functionality.

  • Press s to open user scripts menu
  • Press y to toggle auto-inject on page load
  • Scripts are stored in ~/.brow6el/userscripts/

Example scripts are in the examples/ directory:

  • reader-mode.js - Simplify articles for better readability
  • force-light-mode.js - Override dark themes

JavaScript Console

Press c to toggle the console. View page errors, warnings, and console.log() output. Navigate with j/k, exit with ESC.

Profile Modes

Configure in ~/.brow6el/browser.conf or use command-line flags:

  • --temporary - Private mode, deleted on exit
  • --persistent - Save cookies, cache, history
  • --custom - Use custom profile path

Graphics Protocol

Choose rendering protocol in browser.conf or via command line:

  • --sixel - Use sixel graphics (default, widely supported)
  • --kitty - Use kitty graphics protocol (kitty, ghostty, wezterm)

7. Tips & Tricks

Workflow Optimization

💡 Quick Link Navigation: For pages with many links, use hint mode (f) instead of MOUSE mode. It's much faster!
💡 Form Filling: Use MOUSE mode to click the first field, then press i to enter INSERT mode. Fill the form, then ESC when done.
💡 Reading Articles: Use t and g for smooth scrolling through long articles. Much more comfortable than holding j!
💡 Quick Bookmarking: On a page you like? Just press d. Later press b to access all your bookmarks.

Common Workflows

Search Engine Workflow

  1. Press u to open URL dialog
  2. Type google.com (or your preferred search engine)
  3. Press Enter
  4. When page loads, press i for INSERT mode
  5. Type your search query
  6. Press ESC, then f to click search or first result

Article Reading Workflow

  1. Navigate to article using hint mode or URL
  2. (Optional) Press s and inject reader-mode.js for cleaner layout
  3. Use t/g to scroll smoothly
  4. Press v to select interesting quotes, y to copy
  5. Press d to bookmark if you want to return later

Performance Tips

💡 Graphics Protocol: If using kitty terminal, use --kitty flag for better performance. Otherwise use --sixel (default).
💡 Force Redraw: If the screen looks corrupted, press Z (capital Z) to force a full redraw.

Troubleshooting

⚠️ Terminal not displaying graphics? Make sure your terminal supports sixel (mlterm, xterm, wezterm, foot) or kitty protocol (kitty, ghostty, wezterm).
⚠️ Keys not working? Check your mode! In INSERT mode, navigation keys won't work. Press ESC to return to STANDARD mode.
⚠️ Hint mode not showing hints? Wait a moment for the page to fully render overlays. On complex pages, this might take a second or two.
